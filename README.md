Close Git Combat
========

> Kept You Waiting, Huh? ― *[Big Boss]*<br>
[![i-license]](/LICENSE)
[![i-fish]][fish]
[![i-omf]][omf]

**cgitc** is set of useful aliases to use git much more faster.

```fish
omf install https://github.com/simnalamburt/pkg-cgitc
```

cgitc is fork of oh-my-zsh's [git plugin].
